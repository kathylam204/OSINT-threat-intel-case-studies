# OSINT & Threat Intelligence Case Studies

Welcome to my portfolio of self-directed threat intelligence and open-source intelligence (OSINT) investigations. 

As a Computer Science graduate with a strong foundation in systems, networking, and security, I use these case studies to document real-world threat encounters, analyze malicious infrastructure, and execute operational mitigations.

### Core Methodologies Demonstrated:
* **Infrastructure Analysis:** Investigating lookalike domains, performing passive WHOIS queries, and analyzing DNS records to map out threat actors' setups.
* **OSINT & Entity Attribution:** Cross-referencing public land registries, metadata, server logs, and public data records to uncover composite identities and trace remote actors.
* **Threat Mitigation:** Compiling structured Indicators of Compromise (IoCs) and routing actionable threat briefs to hosting providers and federal agencies like CISA.
